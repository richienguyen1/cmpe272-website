# CMPE272-01 Fictional Company Website
Richie Nguyen, 015533661

## Website Information
- This website is created using HTML and PHP to show the information, products, and users of a factional electronics/plumbing company.
- In addition, the website uses cookies for tracking, CURL calls to other student websites in the course, and MySQL database retrievals.
- A live version of this website is available at https://richienguyencmpe272website.infinityfreeapp.com/index.php.

## Local Installation Using XAMPP
1. Set up the local website ports and MySQL addresses in XAMPP installation
2. Put all subdirectories and .php files in the XAMPP installation at `<User>/xampp/htdocs`
3. Initially run the Apache and MySQL modules in XAMPP Control Panel
4. Enter `localhost/phpmyadmin` on a modern web browser to access the MySQL database
5. Import `setupDatabaseAndTables.sql` in the `resources/mysql` subdirectory in phpMyAdmin

## Running the Local Website After Installation
1. Run the Apache and MySQL modules in XAMPP Control Panel
2. Access the website by entering `localhost/index.php` on a modern web browser
